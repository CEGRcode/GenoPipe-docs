vignette

The ENCODE experiment ENCSR692RET is a ChIP-seq experiment on a K562 cell line sample that has been CRISPR modified with an eGFP insertion to target the NR4A1 gene protein product. This vignette will show how EpitopeID and StrainID can be run on the same sample.

For this example, we are starting from the /User/joeschmoe/Desktop directory. The following code will need to modify this path as appropriate if you run this example on your own machine.

First, download the FASTQ and BAM files for replicate 1 from ENCODE's data portal:

# Get FASTQ (for EpitopeID)
wget https://www.encodeproject.org/files/ENCFF415CJF/@@download/ENCFF415CJF.fastq.gz
wget https://www.encodeproject.org/files/ENCFF368OBR/@@download/ENCFF368OBR.fastq.gz
# Get BAM (for StrainID)
wget https://www.encodeproject.org/files/ENCFF069ERV/@@download/ENCFF069ERV.bam

mkdir DATA

# Rename with experiment accession:
mv ENCFF415CJF.fastq.gz DATA/ENCSR692RET_rep1_R1.fastq.gz
mv ENCFF368OBR.fastq.gz DATA/ENCSR692RET_rep1_R2.fastq.gz
mv ENCFF069ERV.bam DATA/ENCSR692RET_rep1.bam
samtools index DATA/ENCSR692RET_rep1.bam

# Make output directory
mkdir myreports_EID
mkdir myreports_SID

Clone GenoPipe (if you haven't already). As long as you can locate it on your filesystem, it shouldn't matter where you clone it.

git clone https://github.com/CEGRcode/GenoPipe.git

Run EpitopeIDs

# Download Genomic FASTA and move to appropriate directory
cd $WRK/GenoPipe/EpitopeID/utility_scripts/genome_data/
bash download_hg19_Genome.sh
mv genome.fa* ../../hg19_EpiID/FASTA_genome/
cd ../../
# Run EpitopeID
bash identify-Epitope.sh -i /User/joeschmoe/Desktop/DATA/ -o /User/joeschmoe/Desktop/myreports_EID -d /User/joeschmoe/Desktop/GenoPipe/EpitopeID/hg19_EpiID
